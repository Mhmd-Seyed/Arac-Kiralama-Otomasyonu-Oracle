# 🚗 Car Rental Database Automation (Araç Kiralama Otomasyonu)

A comprehensive relational database management system designed to automate car rental operations. Built entirely with **Oracle SQL**, this project focuses on robust data architecture and backend business logic.

## 🛠️ Database Features & Architecture
- **Relational Tables:** Well-structured schema for Vehicles, Customers, Rentals, and Payments.
- **Advanced Queries:** Implementation of complex `JOIN` operations for reporting and data extraction.
- **Stored Procedures:** Encapsulated business logic for processing rentals and calculating costs.
- **Triggers:** Automated actions to maintain data integrity (e.g., updating vehicle availability status automatically).
- **Data Definition (DDL):** Complete scripts for recreating the entire database structure.

## 💻 Tech Stack
- **Database Engine:** Oracle Database
- **Language:** Oracle SQL (PL/SQL)

## 📂 Project Structure
- `Schema.sql`: Contains all DDL commands (Tables, Constraints).
- `Procedures_Triggers.sql`: Contains the business logic and automation scripts.
- `Queries.sql`: Examples of complex queries and reports.
